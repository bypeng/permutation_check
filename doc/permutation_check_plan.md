# 專案名稱：實境遊戲決賽 - 竹簡詩句驗收系統

## 1. 核心功能 (Core Logic)
- **演算法**：Longest Increasing Subsequence (LIS)。
- **目的**：驗證參加者排列的詩句順序，標示出符合正確順序的最長子序列，並將不在序列中的方塊變色（報錯）。

## 2. 介面需求 (UI/UX)
- **排版**：直寫、由右向左排列 (CSS: writing-mode: vertical-rl)。
- **方塊設計**：長條竹片感，顯示詩句前 2-3 個字。
- **操作流程**：
    - 待選區：依照 Unicode/內碼排序，方便關主快速檢索。
    - 排列區：支援拖曳插入 (Insert) 與調整順序。
- **安全性**：簡易密碼登入機制，支援登出後需重新輸入密碼。

## 3. 資料結構 (Data Structure)
- **詩句清單**：40 句含正確序位，另有 0-10 個混淆項。
- **組別管理**：可自訂參賽者隊名/姓名，不使用預設編號。

## 4. 外部整合 (Integration)
- **Line Notify**：點擊驗收成功後，自動發送包含隊名、挑戰次數、通關時間的訊息。
- **部署平台**：Google Apps Script (GAS) 網頁應用程式，嵌入於 Google Sites。
